# MathTools Offline

Uma suíte open source para matemática, lógica formal e resolução simbólica, projetada para funcionar **100% offline**.

## Visão Geral

O MathTools Offline reúne ferramentas essenciais para estudantes, pesquisadores e educadores em uma única aplicação leve e independente da internet.

O projeto tem como objetivo oferecer um ambiente integrado para:

* escrita matemática em LaTeX;
* computação algébrica simbólica;
* verificação formal de demonstrações;
* explicações guiadas de conceitos matemáticos;
* exportação de materiais em PDF.

## Funcionalidades

### Editor LaTeX

* Edição de expressões e documentos matemáticos;
* Visualização em tempo real;
* Suporte a templates acadêmicos;
* Exportação para PDF.

### SymPy

* Simplificação algébrica;
* Derivadas e integrais;
* Resolução de equações;
* Manipulação simbólica;
* Álgebra linear.

### Lean 4

* Escrita e verificação formal de provas;
* Integração com lógica matemática;
* Desenvolvimento assistido de demonstrações.

### Explicador Local

* Explicações passo a passo executadas localmente;
* Funcionamento sem conexão com a internet;
* Foco educacional;
* Linguagem acessível para diferentes níveis de aprendizagem.

### Exportação PDF

* Geração de listas de exercícios;
* Criação de apostilas;
* Impressão de demonstrações e resoluções;
* Compartilhamento de materiais produzidos.
